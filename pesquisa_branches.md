**BRANCH:**

Branch é quando você tira uma copia do código para você conseguir trabalhar com ela em casa e outros lugares sem correr o risco de perder o código, você não mexe na versão principal do código.



**PARA OQUE SERVE:**

O branch serve para que os desenvolvedores teste suas ideias sem medo e sem nenhum risco, podem fazer mudanças, corrigir problemas, mesmo  assim o código principal continuará intacto sem nenhum problema, quando estiver funcionando é sô juntar tudo ao projeto principal



**APLICAÇÕES PRATICAS:**

**1-**Imagina você precisa mudar um código em uma tela inicial, porem para não correr o risco de dar merda, você salva um branch e muda apenas o branch



**2-**Acham um erro em uma tela de login, porem para a tela ainda continuar no ar e não ter o risco de perder o site você, baixa o branch para não correr nenhum risco



**3-**Num trabalho em grupo as pessoas baixam um branch para cada um poder mexer na sua casa e reunir o trabalho no final no projeto principal sem se atrapalharem entre si
